# 📖 天行者 使用說明

> 網址：**https://kindms.github.io/bibleduo/**（手機、電腦都能玩）

## 開始玩（30 秒上手）

1. 用手機打開上面的網址
2. 點右上角 **👤** 用 Google 帳號登入（進度會存雲端，換手機不會消失；不登入也能玩，但進度只存在這支手機）
3. 選一卷書（新手推薦**馬可福音**）→ 點第 1 章 → 開始闖關！

## 玩法

- 每關約 8～10 題：**經文填空、排順序、上下句配對、接下句**，部分書卷還有**理解題**（答完會告訴你依據哪節經文）
- 答對得 ⭐ 經驗值，全對有加碼；答錯扣 ❤️ 愛心（共 5 顆）
- **愛心用完別緊張**：把這一章的經文讀完一遍，就回一顆愛心繼續闖！
- 一關答完如果有答錯的題目，會進入**錯題複習**再練一次——複習答錯不扣愛心
- 每天玩就累積 🔥 連續天數——讓讀經變成每天的習慣！
- 過了這一章才會解鎖下一章；每卷書都可以自由開始

## 好玩的功能

| 按鈕 | 功能 |
|------|------|
| 🏆 | 排行榜：本週榜（每週一歸零）＋總榜，看看小組誰讀最多 |
| 🎨 | 換場景（青草地／加利利海／曠野日出／星夜應許）和夥伴（鴿子／魚／河馬／螞蟻） |
| ✏️（排行榜內） | 改排行榜顯示的名字（也可以直接點榜上自己那一列） |
| 🔊（關卡內） | 音效開關，在教會或圖書館玩可以切靜音 |
| 🚩（答題後） | 覺得題目怪怪的？一鍵回報給管理者 |

## 把它變成手機 App（加入主畫面）

- **iPhone（Safari）**：打開網址 → 點下方「分享」按鈕 → 往下找「**加入主畫面**」→ 完成！
- **Android（Chrome）**：打開網址 → 右上角「⋮」→「**加入主畫面**」→ 完成！

之後從主畫面點圖示，就像開一般 App 一樣——有自己的綠色書本圖示、開起來全螢幕，而且**玩過的經文沒網路也能繼續玩**（捷運上、飛機上都行）。

## 常見問題

- **進度不見了？** 確認有登入 Google（右上角是你的大頭貼就是登入狀態）
- **換手機？** 新手機打開網址、登入同一個 Google 帳號，進度自動回來
- **沒聲音？** 部分手機需要先點過畫面一次才會有音效
